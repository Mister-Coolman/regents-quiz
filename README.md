# NY State Regents Prep AI

A web app that helps students prepare for New York State Regents exams using AI. Users can generate practice questions, list available topics, and count questions by subject or type. Features include PDF downloads and interactive quiz mode.

## Features

- AI-powered query parser using FireworksAI Ollama + Llama3
- Practice questions served from a SQLite database
- Question images displayed and embedded in downloadable PDFs
- Interactive quiz experience
- Flask + SQLite backend (Fly.io)
- React + Vite frontend (Netlify)

## Deployment

### Frontend (Vite + React)

Hosted on **Netlify**
